# Docx to PDF Converter

A Java web application that converts DOCX files to PDF format using asynchronous background processing.

## Prerequisites

-   Docker

## Build and Run with Docker

1. Clone the repository and navigate to the project directory.

```bash
git clone https://github.com/neyugn261/docx-to-pdf.git
cd docx-to-pdf
```

2. Build and start the application:

```bash
docker-compose up --build
```

3. Access the application at `http://localhost:8386/`
